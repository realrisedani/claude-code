Essential Commands

  • /help — alle Commands
  • /init — CLAUDE.md generieren
  • /clear — Context reset
  • /memory — persönliche Regeln
  • /cost — Token-Verbrauch
  • /config — Einstellungen

Power Commands

  • /mcp — MCP-Server verwalten
  • /hooks — Automatisierungen
  • /code-review — Diff analysieren
  • /security-review — Sicherheits-Check
  • /verify — Änderung testen
  • /loop — Recurring Tasks

Custom Command Struktur

  • .claude/commands/standup.md
  • Frontmatter: description + allowed-tools
  • Prompt-Body mit Anweisungen
  • Wird zu /standup im Chat

$ARGUMENTS Beispiel

  • /deploy production
  • → $ARGUMENTS = "production"
  • Im Prompt: "Deploy to $ARGUMENTS"
  • Macht Commands flexibel & wiederverwendbar
