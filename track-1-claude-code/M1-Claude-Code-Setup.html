5 Vorteile vs Browser

  • Datei-Ops direkt im Filesystem
  • Multi-File Kontext gleichzeitig
  • Bash Commands ausführen
  • Git nativ integriert
  • CLAUDE.md Projekt-Gedächtnis

Installation per OS

  • Mac: brew install node → npm i -g @anthropic-ai/claude-code
  • Windows: winget install node → npm i -g
  • Linux: nvm install lts → npm i -g

3 Permission-Modi

  • Default: fragt bei jeder Aktion — sicher
  • Accept-Edits: schreibt automatisch — schnell
  • Plan: nur lesen, nie schreiben — sicher

5 erste Commands

  • /help — alle Commands anzeigen
  • /init — CLAUDE.md generieren
  • /clear — Context zurücksetzen
  • /cost — Token-Verbrauch prüfen
  • /exit — Session beenden
💡 Erste echte Session: Ordner erstellen → claude starten → "Erstelle hello.js" → node hello.js ausführen lassen
