5 Hook-Typen

  • PreToolUse — vor jedem Tool
  • PostToolUse — nach jedem Tool
  • UserPromptSubmit — bei Eingabe
  • Stop + SessionStart

Killer-Setups

  • PostToolUse:Edit → Prettier auto
  • PostToolUse:Edit → tsc --noEmit
  • Stop → Notification (macOS)
  • SessionStart → git status anzeigen

Hook-Variablen

  • $CLAUDE_TOOL_NAME — Tool-Name
  • $CLAUDE_TOOL_INPUT_PATH — Pfad
  • $CLAUDE_USER_PROMPT — Prompt
  • Shell-Skripte vollständig nutzbar

Hook vs CLAUDE.md

  • Hook = deterministisch (immer)
  • CLAUDE.md = Claude-Ermessen
  • Formatting → Hook nicht CLAUDE.md
  • Stil-Hinweise → CLAUDE.md
⚠ Achtung: Hooks die fehlschlagen, blockieren Claude. Immer Fallback: || true am Ende des Commands
