Abschnitt 1 — Installation (5 Min)
# 1. Node.js prüfen (min v18)
node --version

# 2. Claude Code installieren
npm install -g @anthropic-ai/claude-code

# 3. Authentifizieren
claude
# → Browser öffnet sich → mit Anthropic-Account einloggen

# 4. Verifizieren
claude --version
💡 Falls npm fehlt: Node.js von nodejs.org installieren — wähle die LTS-Version (v18 oder höher)
Abschnitt 2 — Die 3 Permission-Modi
🟢
Default
(Standard, kein Flag)
Claude fragt bei jeder Datei-Änderung und jedem Bash-Befehl nach. Du behältst volle Kontrolle.
Wann nutzen:
Immer am Anfang, bei unbekannten Projekten, wenn du neu bist.
"Sicher aber langsam"
🟡
Accept Edits
Shift+Tab (toggle)
Datei-Änderungen werden automatisch akzeptiert. Bash-Befehle werden noch gefragt.
Wann nutzen:
Wenn du dem Projekt vertraust und der git-Stand sauber ist.
"Schnell aber nur mit sauberem git"
⚠ NIEMALS ohne: git status zeigt 0 Änderungen
🔴
Auto / YOLO
claude --dangerously-skip-permissions
Alles automatisch — kein Nachfragen bei Dateien oder Befehlen.
Wann nutzen:
Nur für Wegwerf-Projekte oder wenn du genau weißt was passiert.
⚠ Kann dein Projekt überschreiben. Immer git commit vorher.
Abschnitt 3 — Erste Datei erstellen (Hands-on)
1
Terminal öffnen und in einen Ordner navigieren:
cd ~/Desktop && mkdir mein-projekt && cd mein-projekt
2
Claude Code starten:
claude eingeben → Enter
3
Erste Aufgabe eingeben:
Erstelle eine hallo.html mit 'Hallo Welt' als Überschrift
4
Claude zeigt eine Vorschau der geplanten Datei → Enter zum Bestätigen
5
Datei öffnen und Ergebnis sehen:
open hallo.html (Mac) oder start hallo.html (Windows)
✅ Du hast Claude Code installiert und deine erste Datei erstellt!
