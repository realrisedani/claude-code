# Aufnahme-Methode — gilt für ALLE Tracks

Die eine Methode, nach der jede Lektion aufgenommen wird. Kein Rätselraten mehr.

---

## 1. Die Grundregel: Track 0 vs. der Rest

| | Track 0 | Track 1 und höher |
|---|---|---|
| Deine Rolle | An die Hand nehmen | Schulterblick |
| Du zeigst | Jeden einzelnen Klick | Das Muster, EINMAL sauber |
| Praxis-Anteil | ~70 % | ~40 % |
| Dein Satz | "Mach genau das nach" | "Das kennst du — hier das Profi-Level" |

**Goldene Regel:** Was in Track 0 schon geklickt wurde (Installieren, Git, erste Datei),
in Track 1 NICHT nochmal klicken. Nur kurz benennen, dann das Neue zeigen.

---

## 2. Die 3 Lektions-Typen

Jede Lektion ist genau einer von drei Typen. Der Typ entscheidet über Praxis & Länge.

### 🟢 RECAP — "Kennst du schon"
Thema kam in Track 0 vor.
- **Praxis:** keine
- **Länge:** 5–8 Min
- **Ablauf:** kurz erinnern → nur das Profi-Extra obendrauf zeigen
- **Beispiel:** Installation (Modi sind neu, Install nicht)

### 🔵 WERKZEUG — "Neues Tool"
Ein neues Feature/Tool, das man anfassen muss.
- **Praxis:** EINE saubere Live-Demo
- **Länge:** 12–18 Min
- **Ablauf:** Hook → 1× live vormachen → Student macht es nach
- **Beispiel:** CLAUDE.md, Sub-Agents, Hooks, Worktrees

### 🟣 DENKEN — "Neues Mindset"
Reines Konzept, nichts zum Klicken.
- **Praxis:** keine
- **Länge:** 8–12 Min
- **Ablauf:** über die Slides reden, evtl. 1 Mini-Beispiel
- **Beispiel:** Mental-Model, Cost & Token Economy, Context-Management

---

## 3. Der Aufnahme-Ablauf pro Lektion (immer gleich)

```
1. HOOK (1–2 Min)    — Welcher Schmerz wird heute gelöst? Eine echte Situation.
2. KERN              — RECAP: erinnern · WERKZEUG: 1 Demo · DENKEN: Slides erklären
3. MITMACHEN         — nur bei 🔵 WERKZEUG: "Jetzt du — Pause, mach es nach"
4. ABSCHLUSS (1 Min) — 1 Satz Kernbotschaft + Brücke zur nächsten Lektion
```

---

## 4. Die Demo-Spar-Regel (gegen Wiederholung)

Track 1 ist voll mit 🔵 Werkzeug-Lektionen. Wenn du in jeder alles vollständig
vorklickst, wird es zäh. Deshalb:

- **Erstes Mal** ein Mechanismus vorkommt (Datei anlegen + committen, Befehl
  starten, settings.json öffnen) → **voll zeigen**.
- **Jedes weitere Mal** → durchziehen: *"Den Ablauf kennst du — hier nur das Neue."*

So bleibt jede Lektion frisch, ohne dass du Inhalt weglässt.
