Die 4 Schritte

1

Claude Code installieren Terminal

Einmalig installieren — danach steht claude global zur Verfügung.

npm install -g @anthropic-ai/claude-code
claude --version# Erfolg: zeigt Versionsnummer
💡 Fehler "permission denied"? Dann: sudo npm install -g @anthropic-ai/claude-code — oder Node.js Version prüfen (mind. v18).
2

Authentifizieren Auth

Beim ersten Start fragt Claude Code nach deinem Account. Zwei Optionen:

claude# startet Auth-Flow automatisch

Option A — Claude Pro

  • claude.ai/pro — $20/Mo
  • Im Terminal: "Login with Claude.ai"
  • Browser öffnet sich automatisch
  • Einfachster Start

Option B — API Key

  • console.anthropic.com → API Keys
  • Key kopieren
  • export ANTHROPIC_API_KEY=sk-...
  • Flexibel, pay-per-use
3

Erste HTML-Seite generieren Magie

Ordner erstellen, Claude starten, Prompt eingeben — Claude schreibt die Datei direkt.

mkdir mein-projekt && cd mein-projekt
claude# Session starten
Dein erster Prompt:
"Erstelle eine moderne index.html mit einem großen Titel, kurzer Beschreibung und einem blauen Call-to-Action-Button."

Claude erstellt index.html direkt auf deiner Festplatte — kein Copy-Paste nötig.

open index.html# öffnet im Standard-Browser (Mac)
start index.html# Windows
4

Erste Iteration — Button orange Iteration

Das ist der entscheidende Unterschied zu Browser-Claude: Claude ändert die echte Datei. Kein Copy-Paste.

Vorher → Nachher
Button → "Mach den Button orange" → Button
Im Terminal (Claude läuft noch):
"Mach den Button orange."
→ Claude ändert die Datei → Browser neu laden → fertig.

Was Claude tut

  • Liest index.html automatisch
  • Findet den Button-CSS
  • Ändert background zu #F97316
  • Speichert die Datei direkt

Dein Flow

  • Prompt tippen
  • Claude arbeitet (~3 Sek)
  • Browser Cmd+R neu laden
  • Änderung sehen ✓
Checkliste — hast du alles gemacht?
  • ✓

    claude startet ohne Fehler

🎯 Lernziel erreicht: Du hast Claude Code installiert, authentifiziert, eine HTML-Seite per Sprache erstellt, im Browser geöffnet und deine erste Iteration durchgeführt. Das ist der vollständige Claude-Code-Loop.

Nützliche Befehle

  • /help — alle Befehle
  • /clear — Context reset
  • Esc — Eingabe abbrechen
  • /exit — Session beenden
