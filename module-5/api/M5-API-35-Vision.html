Lernziele

  • Ein Bild als image-content-Block (base64 oder url) an Claude schicken
  • Mehrere Bilder mit Text in einer Nachricht kombinieren
  • Bildgröße und Token-Kosten im Blick behalten
Konzept — Bild als content-Block
💡 Der Trick: content ist nicht nur Text. Du übergibst eine Liste aus Blöcken — ein image-Block plus ein text-Block mit deiner Frage. Claude sieht das Bild und antwortet darauf.
Bild per base64 senden
import base64
from anthropic import Anthropic
client = Anthropic()

with open("screenshot.jpg", "rb") as f:
  img_b64 = base64.standard_b64encode(f.read()).decode("utf-8")

resp = client.messages.create(
  model="claude-sonnet-4-5", max_tokens=1024,
  messages=[{
    "role": "user",
    "content": [
      {
        "type": "image",
        "source": {
          "type": "base64",
          "media_type": "image/jpeg",
          "data": img_b64,
        },
      },
      {"type": "text", "text": "Was zeigt dieser Screenshot?"},
    ],
  }],
)
print(resp.content[0].text)
✅ media_type muss passen: image/jpeg, image/png, image/gif oder image/webp — exakt zum echten Dateiformat. Ein PNG als jpeg deklariert führt zu Fehlern.
Bild per URL senden
{
  "type": "image",
  "source": {
    "type": "url",
    "url": "https://example.com/diagramm.png",
  },
}
🚫 Bildgröße = Token-Kosten: Große Bilder kosten viele Tokens. Sehr große Bilder werden zudem herunterskaliert. Verkleinere vorab auf eine vernünftige Auflösung — das spart Geld und ändert das Ergebnis meist nicht.
Mehrere Bilder + Text mischen
"content": [
  {"type": "text", "text": "Vergleiche diese beiden Designs:"},
  {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": a}},
  {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": b}},
  {"type": "text", "text": "Welches wirkt professioneller?"},
]

Szenario — Rechnung auslesen

Szenario: Foto einer Rechnung in strukturierte Daten

1
Rechnungsfoto einlesen, mit base64.standard_b64encode kodieren
2
image-Block + text-Block "Gib mir Betrag, Datum und Anbieter als JSON"
3
media_type exakt setzen (z. B. image/jpeg)
4
Antwort aus resp.content[0].text als JSON weiterverarbeiten
Tipp für saubere Daten
Sag Claude im text-Block genau, welches Format du willst ("als JSON mit den Feldern betrag, datum, anbieter") — dann ist die Antwort direkt maschinenlesbar.
💡 Jetzt du: Nimm einen beliebigen Screenshot, kodier ihn nach base64 und schick ihn mit der Frage "Beschreibe in 3 Stichpunkten, was hier zu sehen ist". Achte darauf, dass media_type zum echten Dateityp passt.
