Lernziele

  • Einen Stream mit client.messages.stream(...) starten und Text live ausgeben
  • Die wichtigsten SSE-Events kennen (message_start, content_block_delta, message_stop)
  • Verstehen, warum Streaming die gefühlte Geschwindigkeit drastisch verbessert
Konzept — Warum streamen
💡 Ohne Streaming wartet der User, bis die KOMPLETTE Antwort fertig ist — bei langen Texten zehn Sekunden Leere. Mit Streaming erscheint Wort für Wort, sofort. Die Rechenzeit ist dieselbe, aber die App fühlt sich um ein Vielfaches schneller an.
Der einfache Weg — text_stream
from anthropic import Anthropic
client = Anthropic()

with client.messages.stream(
  model="claude-sonnet-4-5",
  max_tokens=1024,
  messages=[{"role": "user", "content": "Schreib eine kurze Geschichte."}],
) as stream:
  for text in stream.text_stream:
    print(text, end="", flush=True) # jedes Häppchen sofort

  final = stream.get_final_message() # vollständige Nachricht + usage
  print()
  print(final.usage.output_tokens)
✅ Faustregel: Für die meisten Apps reicht stream.text_stream — du bekommst direkt die Text-Häppchen. get_final_message() liefert am Ende die komplette Nachricht inkl. usage fürs Cost-Logging.
Die rohen Events (wenn du mehr Kontrolle brauchst)
with client.messages.stream(
  model="claude-sonnet-4-5", max_tokens=1024,
  messages=[{"role": "user", "content": "Hallo!"}],
) as stream:
  for event in stream:
    if event.type == "content_block_delta":
      print(event.delta.text, end="", flush=True)
    elif event.type == "message_stop":
      print("\n[fertig]")
Die wichtigsten Events
EventFeldBedeutung
message_start—Antwort beginnt
content_block_deltadelta.textein Text-Häppchen
content_block_stop—Block fertig
message_stop—Antwort komplett
🚫 Falle: Nicht jedes Event enthält Text. Wenn du blind event.delta.text liest, fliegt dir ein Fehler um die Ohren. Prüf immer erst event.type == "content_block_delta".

Warum es wirkt
Erste sichtbare Wörter nach Sekundenbruchteilen statt nach 8 Sekunden Stille — das ist der ganze Unterschied zwischen "hängt" und "reagiert sofort".
💡 Jetzt du: Schreib ein Skript, das eine lange Antwort streamt (z. B. "Erklär mir Photosynthese in 200 Wörtern"). Gib jedes Häppchen mit print(text, end="", flush=True) aus und logge am Ende über get_final_message().usage die Output-Tokens.
