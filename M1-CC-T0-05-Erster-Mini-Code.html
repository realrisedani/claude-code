Was wir bauen

  • Name ausgeben — console.log("Hey!")
  • Uhrzeit anzeigen — new Date()
  • Random-Spruch — Array + Math.random
  • 3 Zeilen Code reichen

Die 3 Schritte

  • mkdir hallo-projekt — Ordner erstellen
  • code . — VS Code öffnen
  • hallo.js — neue Datei anlegen
  • node hallo.js — ausführen

Code-Zeilen erklärt

  • const — Variable speichern
  • console.log — Text ausgeben
  • new Date() — aktuelle Uhrzeit
  • Math.random() — Zufallszahl

Variationen zum Üben

  • Namen ändern — eigener Text
  • Sprüche ergänzen — Array erweitern
  • Emojis hinzufügen — einfach reinschreiben
  • chalk — Farben im Terminal
✅ Fehler sind normal! SyntaxError = Tippfehler. ReferenceError = Variable falsch geschrieben. Einfach lesen + fixen.
