# Claude Code Kurs — RealRise

Das vollständige Lernmaterial für Claude Code — von Null bis Produktiv.

## Struktur

```
claude-code/
├── track-0-onramp/        — Vorkenntnisse: KI-Landschaft, Git, Terminal
├── track-1-claude-code/   — Claude Code Kern: Installation bis Deploy
├── module-5/
│   ├── api/               — Anthropic Messages API
│   └── agency/            — Agency-Betrieb mit Claude
├── _templates/            — Vorlagen für neue Lektionen
├── _demo/                 — Beispiel-Outputs (nicht für Produktion)
└── .claude/               — Claude Code Konfiguration für diesen Kurs
```

## Tracks

### Track 0 — Onramp (T0)
Für absolute Einsteiger. KI-Landschaft, Kosten, Terminal-Basics, Git in 5 Min, erste Magic-Momente.

### Track 1 — Claude Code (CC)
Der Hauptkurs. Permission-Modi, CLAUDE.md, Memory, MCP, Hooks, Skills, Custom Commands, Git-Workflow, Code Review, Deploy.

### Module 5 — Vertiefung
- **API**: Messages API, Tool Use, Streaming, Vision, Files API
- **Agency**: Client Onboarding, Pricing, Scale

## Lektions-Benennung
`M[Modul]-[Track]-[Nummer]-[Thema].html`

Beispiel: `M1-CC-03-Permission-Discipline.html`
- M1 = Modul 1
- CC = Track: Claude Code
- 03 = Lektion 3
- Permission-Discipline = Thema
