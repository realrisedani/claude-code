Was MCP löst

  • Claude kann nur Text nativ
  • MCP: GitHub Issues/PRs steuern
  • MCP: Notion Pages schreiben
  • MCP: Browser + DB anbinden

Setup Filesystem

  • settings.json → mcpServers
  • npx @modelcontextprotocol/server-filesystem
  • Pfade als args übergeben
  • Claude liest Files außerhalb Projekt

GitHub MCP

  • npx @modelcontextprotocol/server-github
  • GITHUB_TOKEN in env eintragen
  • Issues, PRs, CI direkt aus Claude
  • Kein Browser-Wechsel mehr nötig

MCP vs Alternativen

  • MCP = externe Tools anbinden
  • Sub-Agent = isolierter Claude
  • Custom Command = Prompt-Vorlage
  • Hook = deterministisch, lokal
💡 Start: Filesystem-MCP → sofort Files außerhalb Projekt lesen. Dann GitHub → Issues/PRs direkt aus Claude
