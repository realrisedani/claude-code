Terminal öffnen

  • Mac — Cmd+Space → "Terminal"
  • Windows — Win+X → Terminal
  • Linux — Strg+Alt+T
  • VS Code — Strg+` (integriert)

5 Terminal-Befehle

  • pwd — wo bin ich?
  • ls — was ist hier?
  • cd ordner — wechseln
  • mkdir name — Ordner erstellen

VS Code

  • Download — code.visualstudio.com
  • Sidebar — Datei-Explorer links
  • Terminal — View → Terminal
  • Extensions — Marketplace durchsuchen

Node + bun installieren

  • Node.js — nodejs.org downloaden
  • bun — curl -fsSL bun.sh/install | bash
  • Test Node — node --version
  • Test bun — bun --version
💡 Test: node --version und bun --version sollten eine Versionsnummer zeigen — nicht "command not found"
🎯 Lernziel: Terminal offen, VS Code installiert, node + bun laufen — bereit für Code
