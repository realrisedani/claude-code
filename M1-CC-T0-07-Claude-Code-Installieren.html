Installation

  • npm install -g @anthropic-ai/claude-code
  • claude --version — Erfolg prüfen
  • Bei Fehler — sudo npm install -g
  • Alternative — bun add -g

Auth-Optionen

  • Pro $20/Mo — einfachster Start
  • API-Key — flexibel, pay-per-use
  • API-Key holen — console.anthropic.com
  • ANTHROPIC_API_KEY — env variable

Erste Session

  • claude — im Terminal starten
  • "Erstelle index.html Willkommensseite"
  • open index.html — Browser öffnen
  • Claude schreibt direkt in Dateien

4 Start-Befehle

  • /help — Übersicht aller Befehle
  • /clear — Context zurücksetzen
  • /exit — Session beenden
  • Ctrl+C — Abbrechen
